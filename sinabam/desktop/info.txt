Пример запрос (то, что должен передать js в py):

{
    "index_user": "provod", (логин юзера)
    "index_action": 0, (индекс действия)
    "index_text": "provod_0", (индекс текста, который пользователь хочет отредактировать или удалить, если пользователь захочет отредактировать или удалить свою запись)
    "text": "KFU top 1", (текст, если пользователь хочет его добавить)
}

Индексы действий при запросе к бд:

0 - авторизация
1 - лог аут
2 - добавить запись
3 - удалить запись
4 - отредактировать текст
